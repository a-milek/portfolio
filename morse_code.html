Zephyr Morse Code Transmitter

Purpose: Translation and transmition of morse code through radio

The program is intended to convert keyboard input into Morse code and subsequently transmit it through one of the outputs, providing the ability to connect the board to a radio for further communication. It is also my first Zephyr project so I also used it as an introduction to this Operating System.

Language used: C in Zephyr
