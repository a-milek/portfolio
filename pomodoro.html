Pomodoro App

An app that i created as a part of my Engineerings thesis. It's A web application designed to improve focus and productivity by implementing the Pomodoro technique, featuring a customizable timer, integration of ambient sounds from YouTube, and a minimalist interface created with React and Chakra UI. The app stores user settings locally using localStorage and has been thoroughly tested for functionality and compatibility across various browsers and devices.

Pomodoro is a technique created by Francesco Cirillo designed to improve focus and productivity. It involves breaking work into intervals, typically 25 minutes of focused activity (called a "Pomodoro"), followed by a short break of 5 minutes. After completing four Pomodoros, a longer break of 15–30 minutes is taken. This cycle helps manage mental fatigue, maintain concentration, and enhance work efficiency.

Framework used: React and Robot Framework for testing purposes

You can test it here: https://a-milek.github.io/pomodoro-app/
