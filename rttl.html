RTTL Music Player

Purpose: Importing and Playing music from RTTL format using a DOS 8086 emulator, Real Time Clock, and PC Speaker.

Utilizing command line arguments, you can designate a file name containing a song in RTTL format. The program will then proceed to open the file and retrieve the song title, which will be displayed on the screen. It will import default music settings such as note length (d), default octave (o), and default tempo (b). If these parameters are not explicitly defined alongside the notes, the program will use the default values preset within the program. The program uses the PC Speaker and OUT-assembly instruction to play the notes. Additionally, as the music plays, the program alters the window colors corresponding to the notes being played. During playback, you have the option to adjust the tempo by using the +/- keys on your keyboard.

Language used: Assembly

Environment: DOSBOX emulator

681 Lines of code!

RTTL Format in a txt file:

Cantina:o=5,d=8,b=250,b=250:a,p,d6,p,a,p,d6,p,a,d6,p,a,p,g#,4a,a,g#,a,4g,f#,g,f#,4f.,d.,16p,4p.,a,p,d6,p,a,p,d6,p,a,d6,p,a,p,g#,a,p,g,p,4g.,f#,g,p,c6,4a#,4a,4g
