Vending Machine Restoration

"The project I'm currently working on is by far one of the most complex I've ever taken part in. The goal is to modernize an old vending machine by replacing its physical buttons with a touchscreen interface and a companion app, which I’ve developed using React. Another major component involves building an LCD sniffer using an Arduino Nano that captures data from an old Hitachi HD44780 display and transmits it via WebSocket(in progress). This project is highly advanced and has given me the opportunity to rapidly develop a wide range of both

Frameworks and Languages used: React, Python, Arduino

Github Repository(LCD Sniffer): https://github.com/a-milek/LCD-Arduino-Sniffer

Github Repository(App and servers): https://github.com/a-milek/VendingMachine
